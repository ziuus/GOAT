# GOAT Agent Template Marketplace

Agent Templates are predefined setups for subagents or Studio agents. They define:
- Role & Purpose
- Model Recommendation
- Required Tools
- Instructions & Examples

## Integration
Agent templates can be discovered via the remote marketplace or locally defined.
They can be imported directly into the AI Studio Agent Builder.

## Tool Safety
Agent templates only *request* tools. Upon creation, the user must approve any dangerous tools.
