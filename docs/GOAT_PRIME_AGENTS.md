# GOAT Prime Agents

Prime Agents are the highest tier of intelligent entities in the GOAT ecosystem. They own large domains of responsibility, coordinate complex workflows, and manage Specialist Agents.

## Built-In Prime Agents

### 1. Cofounder
- **Status:** Planned / Experimental
- **Domain:** Idea validation, market research, MVP planning, weekly founder reports.
- **Philosophy:** Evidence-first thinking. Prevents premature building.
- **Specialists:** Finance Analyst, Unit Economics Analyst, Market Validator, Pricing Strategist.

### 2. Socializer
- **Status:** Planned
- **Domain:** Personal branding, content strategy, launch planning, community engagement.
- **Philosophy:** Ethical distribution. No spam automation. Generates drafts for human approval.
- **Specialists:** Reddit Strategist, LinkedIn Writer, X Writer, Community Manager.

### 3. Designer
- **Status:** Planned
- **Domain:** UI/UX review, layout critique, design systems, accessibility checks.
- **Philosophy:** Visual and copy hierarchy over generic templates.
- **Specialists:** UI Critic, Accessibility Reviewer, Landing Page Reviewer.

### 4. Researcher
- **Status:** Experimental
- **Domain:** Deep web research, source checking, trend analysis, paper summarization.
- **Philosophy:** Comprehensive, cited, and unbiased information gathering.
- **Specialists:** Source Checker, Trend Analyst, Technology Comparator.

### 5. Builder
- **Status:** Experimental / Active (maps to current coding capabilities)
- **Domain:** Implementation planning, repo inspection, coding, tests, docs.
- **Philosophy:** Robust, secure, and maintainable code generation.
- **Specialists:** Code Reviewer, Test Writer, Refactorer, Docs Writer.

### 6. Operator
- **Status:** Planned
- **Domain:** Deployment, monitoring, logs, crash analysis, CI/CD.
- **Philosophy:** Reliability and uptime. Predictable rollbacks.
- **Specialists:** DevOps Specialist, Log Analyst, Incident Reviewer.

### 7. Learner
- **Status:** Planned
- **Domain:** DSA, AI/ML, Rust, daily learning plans, revision tracking.
- **Philosophy:** Continuous improvement and mastery.
- **Specialists:** DSA Coach, ML Tutor, Rust Tutor, Exam Prep Coach.

## Custom Prime Agents
Users can define Custom Prime Agents by providing a valid manifest and specifying domain responsibilities. Custom agents integrate automatically into the Dashboard and Command systems.
