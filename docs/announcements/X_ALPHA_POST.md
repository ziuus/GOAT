**Tweet 1 / Thread Start**
Just open-sourced the first Alpha of GOAT: A local-first AI Agent OS. 🐐

It’s a unified system (Rust daemon + Next.js dashboard) for structured, multi-agent workflows—designed for builders, founders, and students.

Repo: [Link]

**Tweet 2**
What works today?
✅ Cofounder Agent: Validates ideas & scopes MVPs.
✅ Learner OS: Auto-generates study roadmaps.
✅ PromptForge: Compiles vague ideas into strict prompts.
✅ ApprovalGate: Blocks terminal automation until you say yes.

**Tweet 3**
Why Local-First?
Agent frameworks are cool, but handing over a root shell to a cloud API is terrifying. GOAT uses a local Rust daemon with an `ApprovalGate` to intercept actions, and stores your project memory locally in SQLite. 

**Tweet 4**
Honesty check: This is an active Alpha. We aren't claiming AGI. The web UI is still catching up to the backend collaboration layer, and some agents are marked experimental.

If you like Rust, Next.js, and local AI, check it out and let me know what breaks! 🤝
