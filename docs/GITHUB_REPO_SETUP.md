## Features & Permissions
* **Issues:** Enabled (Use Issue Templates for structured feedback)
* **Discussions:** Recommended later (Once alpha user base grows)
* **Projects:** Optional (Can use GitHub Projects for Roadmap tracking)
* **Releases:** Enabled (Follow the RELEASE_PROCESS.md guide to publish binaries)

## Security
* Enable **Dependabot Alerts** to catch vulnerable dependencies early.
* Enable **Secret Scanning** to ensure no tokens are leaked accidentally.
* Set up a **Security Policy** pointing to `docs/SECURITY.md` or `.github/SECURITY.md`.
