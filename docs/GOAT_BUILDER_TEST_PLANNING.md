# GOAT Builder Test Planning

Test planning designs validation sequences for code changes.

## Automatic Validation Commands
- **Rust Stack:** Suggests running `cargo check`, `cargo fmt --check`, and `cargo test`.
- **NextJS / JS Stack:** Suggests `npm run lint`, `npm run build`, and `npm test`.
- **Manual Routes:** Mentions specific pages to browse on the local dashboard (e.g. `/builder`, `/jobs`).
