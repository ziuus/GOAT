# GOAT AgentFlow Workflows

## Startup Validation Flow
1. **Cofounder**: Validate Idea
2. **Researcher**: Market Scan
3. **Designer**: Design Critique
4. **Socializer**: Launch Plan
5. **Builder**: Builder Handoff

## Launch Readiness Flow
1. **Cofounder**: Positioning Check
2. **Designer**: Onboarding Check
3. **Socializer**: Launch Calendar
4. **Operator**: Readiness Checklist

## Build and Release Flow
1. **Builder**: Implementation
2. **Operator**: Release Plan
3. **Researcher**: Risk Check

## Learning Project Flow
1. **Learner**: Scope Project
2. **Researcher**: Topic Scan
3. **Builder**: Implementation Handoff
4. **Designer**: UX Review

## Incident Response Flow
1. **Operator**: Incident Intake
2. **Researcher**: Error Investigation
3. **Builder**: Fix Handoff
4. **Operator**: Postmortem
