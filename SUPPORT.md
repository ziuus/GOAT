# Support

GOAT is currently in Alpha development. 

If you encounter issues or bugs:
1. **Check the existing issues** on GitHub to see if your problem has already been reported.
2. **Open a new bug report** using the Bug Report template if your issue is new.
3. Provide as much detail as possible, including the output of `goat doctor`.

For feature requests, please use the Feature Request template.

## Community

Community discussions, Discord channels, and forums will be announced closer to the Beta release. Stay tuned!
