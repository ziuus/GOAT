# GOAT Product North Star

## What GOAT is
GOAT is a unified, local-first AI Agent OS. It transforms abstract project goals into coordinated, autonomous local agent work.

## What GOAT is not
- A simple chatbot or wrapper around a single LLM API.
- A fragmented collection of independent AI tools.
- A cloud service that silently exports user data or requires constant internet connection for local-first operations.
- A black-box system that executes dangerous terminal commands without user oversight.

## Target Users
Developers, makers, founders, and researchers who want the power of a coordinated AI team operating directly on their local machine, within their own environments and files.

## Core Promise
“GOAT turns a project goal into coordinated local agent work, with memory, evidence, approvals, reports, and safe execution visible in one workspace.”

## Main User Loop
1. **Input Goal**: The user opens the Mission Control command center and describes a goal.
2. **Plan & Recommend**: GOAT classifies the goal, recommends a team of Prime Agents and Specialist Agents, and drafts a safe execution plan.
3. **Execute & Coordinate**: The selected agents perform their tasks—researching, building, designing, etc.
4. **Approve Risks**: Any dangerous actions (like executing bash commands) are caught by the ApprovalGate and surfaced immediately to the user for sign-off.
5. **Review Artifacts**: The user inspects reports, builder plans, timelines, and browser artifacts generated during the execution, all within a unified activity feed.

## Safety Promise
No hidden autonomous execution. No risky actions without explicit user approval via ApprovalGate. All agent actions are fully inspectable, auditable, and rollback-ready.

## Local-First Promise
GOAT prioritizes local execution and storage. Data remains on the user's machine. Provider routing allows for external LLMs, but the intelligence orchestration, memory (Brain), and execution happen entirely locally.

## Dream End State
Operating GOAT should feel like managing an incredibly competent, tireless, and transparent local AI team. The user dictates the strategy; GOAT handles the tactics, safely and visibly.

## Alpha Readiness
The public alpha release should *only* happen when this core loop—planning, safe execution, and unified review—works seamlessly and cohesively in the Mission Control workspace.
