# GOAT License & Reuse Matrix

Matrix tracking code and concept reuse safety from researched agents, per the Phase 2.2 Deep Research Report.

## Rules
- **MIT / Apache-2.0**: Generally safe. Still require attribution and license preservation.
- **GPL / AGPL**: High caution; do not copy code to keep GOAT permissive, unless GOAT changes its license.
- **Proprietary**: Ideas/concepts only.
- **Unknown/Unverified**: Do not copy code until verified.
