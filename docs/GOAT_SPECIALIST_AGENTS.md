# GOAT Specialist Agents

Specialist Agents are focused domain experts designed to assist Prime Agents. They do not own broad project strategies but provide deep analysis or execute complex specialized tasks within a defined boundary.

## Attachment Model
A Specialist Agent is typically "attached" to a Parent Prime Agent. When the Prime Agent encounters a problem requiring specific expertise, it routes the sub-task to the Specialist Agent.

## Initial Built-In Specialists

### Under Cofounder
- **Finance Analyst:** Revenue models, burn rate analysis, cash flow projections.
- **Unit Economics Analyst:** LTV, CAC, marginal cost evaluation.
- **Market Validator:** Customer discovery interview synthesis, survey analysis.
- **Competitor Analyst:** Feature matrices, pricing comparisons.
- **MVP Scoper:** Trimming feature bloat for v1.0.
- **Pricing Strategist:** Tier structuring, value-based pricing logic.

### Under Socializer
- **Reddit Strategist:** Subreddit culture analysis, un-promotional framing.
- **LinkedIn Writer:** Professional narrative structuring.
- **X/Twitter Writer:** Hook generation, thread formatting.
- **Community Manager:** Discord/Slack tone matching, engagement ideas.
- **Outreach Writer:** Cold email drafting (review required before sending).
- **Launch Planner:** Product Hunt / Hacker News release coordination.

### Under Designer
- **UI Critic:** Aesthetic audit against Antigravity principles.
- **Accessibility Reviewer:** WCAG compliance, color contrast, ARIA checks.
- **Landing Page Reviewer:** Conversion rate optimization (CRO) heuristics.
- **Design System Reviewer:** Token consistency, component reuse.

### Under Researcher
- **Source Checker:** Fact-checking and citation verification.
- **Trend Analyst:** Data aggregation on emerging technologies.
- **Paper Summarizer:** Academic research distillation.
- **Technology Comparator:** Framework A vs Framework B deep dives.

### Under Builder
- **Code Reviewer:** Security, performance, and style analysis.
- **Test Writer:** Unit, integration, and e2e test generation.
- **Refactorer:** Codebase consolidation and architecture boundary enforcement.
- **Docs Writer:** README, API reference, and JSDoc generation.
- **Release Engineer:** Changelog generation, semantic versioning checks.
- **Database Architect:** Schema design, migration planning, query optimization.

### Under Operator
- **DevOps Specialist:** Infrastructure as Code (IaC) generation, Dockerizing.
- **Log Analyst:** Identifying anomalies in structured log data.
- **Incident Reviewer:** Post-mortem drafting, root cause analysis (RCA).
- **CI/CD Engineer:** GitHub Actions workflow generation and optimization.

### Under Learner
- **DSA Coach:** Algorithm pattern recognition, time complexity analysis.
- **ML Tutor:** Core concepts in AI, model architectures.
- **Rust Tutor:** Borrow checker, lifetimes, concurrency patterns.
- **Web3 Tutor:** Smart contracts, consensus mechanisms.
- **Exam Prep Coach:** Spaced repetition schedule generation.

## Future Extensibility
New Specialist Agents can be added via the AI Studio. For example, a user could create an "SEO Analyst" and attach it to the "Socializer" Prime Agent.
