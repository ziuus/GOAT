# GOAT Memory Galaxy

## Overview
Memory Galaxy is the visual manifestation of GOAT's learned knowledge base. It provides a dashboard interface (accessible via the desktop app or web dashboard at `/memory`) to explore everything GOAT has learned about your project, your preferences, and your workflows.

## Features
1. **Candidate Review**: A dedicated inbox for pending memory candidates. You can quickly accept or reject propositions.
2. **Knowledge Graph Clusters**:
    - **User**: Your personal preferences and style guidelines.
    - **Project**: Contextual facts about the repository, stack, and architecture.
    - **Skills**: Reusable tools, scripts, and prompts.
    - **Workflows**: Common automated sequences.
    - **Decisions**: Historical architectural choices and why they were made.
    - **Errors/Solutions**: Previous issues encountered and how they were resolved.
3. **Transparency**: Every memory shows its *Source* (e.g., "Extracted from Job #123") and *Confidence* level.

## Interacting with the Galaxy
- **Accept/Reject**: Click the checkmark or X on pending cards to route them.
- **Filter**: Filter by type, status, or source.
- **Delete**: Even accepted memories can be archived or deleted directly from the dashboard if they are no longer relevant.
